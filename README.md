# Instagram Clone

A mobile Instagram clone built with **Flutter**, **Firebase**, and **Bloc** for state management.  
The app allows users to share photos and videos, like and comment on posts, view stories, follow other users, and interact in real-time.

---

## 🚀 Features

- 🔐 **Authentication & User Profiles**
  - Firebase Authentication (Email / Google login)  
  - Create and edit profile  
  - Role-based access (optional)  

- 🖼 **Posts & Media**
  - Upload photos and videos  
  - Add captions and hashtags  
  - View feed in real-time  

- ❤️ **Engagement**
  - Like posts  
  - Comment on posts  
  - Follow / Unfollow users  

- 📖 **Stories**
  - View other users’ stories  
  - Post your own story (photo/video)  

- ⚡ **Real-time updates**
  - Firestore real-time data sync  
  - Bloc state management  

- 🧩 **Other Features**
  - Search users  
  - Explore feed (optional)  
  - Notifications (optional)  

---

## 🧰 Tech Stack

- **Flutter** – Frontend & UI  
- **Firebase** – Backend (Auth, Firestore, Storage)  
- **Bloc** – State management  
- **Dart** – Programming language
